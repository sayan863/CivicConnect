# CivicConnect

CivicConnect is a community issue reporting and tracking platform designed to improve transparency and prioritization of local civic problems.

## Features
- Report community issues (road, sanitation, safety, etc.)
- Crowd-based priority detection through repeated reports
- Status tracking (Pending / Resolved)
- Time-stamped reports for urgency awareness
- Clean and responsive user interface

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Browser LocalStorage
